# 📈 Bluestock IPO Web Application

A modern FinTech web application built to track and display Initial Public Offerings (IPOs), providing users with detailed company insights, issue information, and investment-related data.

---

## 🚀 Features

* 📊 View all upcoming and ongoing IPOs
* 🏢 Company details with logo and description
* 💰 Price band, issue size, and issue type
* 📅 Open and close dates of IPOs
* 📈 Listing price, CMP, and returns
* 📄 Download RHP / DRHP documents
* 🔍 Clean and responsive UI

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Tailwind CSS
* JavaScript

### Backend:

* REST API (Integrated / External API)
